# FinTrack v1.0.0 - Release Notes

## What's New
- Track income, expenses, and cash transactions
- Import bank statements (CSV, Excel, PDF) from 15+ Indian banks
- AI-powered transaction categorization (on-device)
- Spending analytics with interactive charts
- Monthly budget tracking with progress rings
- UPI transaction parser
- Biometric & PIN app lock
- Hindi language support
- Encrypted backup (.fintrack format)
- Weekly spending notification summaries
- Onboarding flow with account setup
- Dark mode support (coming soon)

## Supported Banks
SBI, HDFC, ICICI, Axis, Kotak, PNB, Bank of Baroda, Canara, Union Bank, IndusInd, IDFC First, Yes Bank, Federal Bank, RBL Bank, South Indian Bank

## Device Requirements
- iOS 16.0+ / Android 8.0+
- Minimum 3GB RAM (basic features)
- 4-6GB RAM recommended (AI features)
- ~50MB storage
